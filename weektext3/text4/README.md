# 题目4：Matplotlib 基础绘图

## 任务要求

请编写一个Python程序，使用 Matplotlib 完成以下绘图任务：

1. 创建 x 轴数据：从 0 到 10，包含 100 个点
2. 计算三组 y 轴数据：
   - y1 = x^2（二次函数）
   - y2 = x^3（三次函数）
   - y3 = 2^x（指数函数）
3. 在同一个图中绘制这三条曲线：
   - y1 使用红色实线，标签为 "y = x²"
   - y2 使用蓝色虚线，标签为 "y = x³"
   - y3 使用绿色点线，标签为 "y = 2^x"
4. 添加标题 "数学函数对比"
5. 添加 x 轴标签 "x"，y 轴标签 "y"
6. 添加图例和网格
7. 保存图像为 `functions_plot.png`

## 要求

- 使用 Numpy 和 Matplotlib 库
- 正确设置线条样式和颜色
- 图表包含所有必要的元素（标题、标签、图例、网格）

## 提示

- 使用 `np.linspace()` 生成 x 数据
- 使用 `**` 运算符计算幂次
- 使用 `plt.plot()` 绘制曲线，注意设置 color、linestyle 和 label 参数
- 使用 `plt.legend()` 显示图例
- 使用 `plt.grid()` 添加网格
