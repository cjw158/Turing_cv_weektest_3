# 题目3：Numpy 数组基础操作

## 任务要求

请编写一个Python程序，使用 Numpy 完成以下数组操作：

1. 创建一个 5x5 的二维数组，元素值从 1 到 25
2. 打印输出该数组
3. 提取并打印以下内容：
   - 第 2 行的所有元素
   - 第 3 列的所有元素
   - 右下角的 2x2 子数组
4. 计算并打印：
   - 整个数组的总和
   - 每一行的和
   - 每一列的和
   - 数组的均值和标准差
5. 将数组转置并打印输出

## 要求

- 使用 Numpy 库完成所有操作
- 正确使用数组索引和切片
- 能够使用 Numpy 的统计函数

## 提示

- 使用 `np.arange()` 和 `reshape()` 创建数组
- 使用数组切片 `array[row, col]` 提取元素
- 使用 `np.sum()` 计算和，注意 axis 参数
- 使用 `np.mean()` 和 `np.std()` 计算统计量
- 使用 `.T` 或 `np.transpose()` 进行转置
