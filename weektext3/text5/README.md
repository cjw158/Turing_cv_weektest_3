# 题目5：Pandas 数据分析基础

## 任务要求

请编写一个Python程序，使用 Pandas 完成以下数据分析任务：

1. 创建一个包含以下数据的 DataFrame：
   - 学生姓名：['张三', '李四', '王五', '赵六', '孙七']
   - 数学成绩：[85, 92, 78, 95, 88]
   - 英语成绩：[90, 85, 82, 88, 95]
   - Python成绩：[88, 90, 85, 92, 87]

2. 打印输出整个 DataFrame

3. 完成以下操作：
   - 计算每个学生的总分，添加为新列 "总分"
   - 计算每个学生的平均分，添加为新列 "平均分"
   - 打印更新后的 DataFrame

4. 数据统计：
   - 打印每门课程的平均分
   - 找出总分最高的学生及其信息
   - 找出平均分最低的学生及其信息

5. 使用 Matplotlib 绘制条形图：
   - x 轴为学生姓名
   - y 轴为总分
   - 添加标题 "学生总分对比"
   - 保存为 `scores_chart.png`

## 要求

- 使用 Pandas 库进行数据处理
- 使用 Matplotlib 进行数据可视化
- 代码要有清晰的注释

## 提示

- 使用 `pd.DataFrame()` 创建数据框
- 使用 `df['列名']` 访问和添加列
- 使用 `.mean()` 计算平均值
- 使用 `.idxmax()` 和 `.idxmin()` 找到最大最小值的索引
- 使用 `plt.bar()` 绘制条形图
